# 🍬 SanabalSweets - E-Commerce Store

A full-stack MERN e-commerce web application for a sweets shop.

## 🔗 Live Demo
https://snabalsweets.vercel.app/

## ✨ Features
- 🛍️ Browse products by category
- 🛒 Add to cart & manage quantity
- 🔐 JWT Authentication (Signup/Login)
- 📦 Place orders
- 📬 Contact form
- 👤 Role-based access (Admin/User)

## 🛠️ Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- Axios
- React Router

**Backend**
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT & Bcrypt
